# 태양광 발전량 데이터셋 품질 확보

## 프로젝트 개요

본 프로젝트는 태양광 발전량 예측을 위한 공공 데이터셋을 대상으로 데이터 품질 확보 과정을 수행하는 프로젝트입니다.

태양광 발전량 데이터는 발전일자, 지역, 기상 변수, 일사량, 설비용량, 발전량 등으로 구성됩니다. 이러한 데이터에는 결측값, 중복값, 물리적으로 불가능한 값, 야간 발전량 불일치 등이 포함될 수 있습니다.

본 프로젝트에서는 태양광 발전량 예측 모델을 학습하기 전에 데이터의 유일성, 완전성, 유효성, 일관성, 정확성을 확보하는 전처리 과정을 수행합니다.

## 데이터셋 정보

## 주요 처리 내용

### 1. 유일성 확보

* `시도명`, `발전일자` 기준으로 중복 데이터를 제거합니다.
* 중복 제거 전후 행 수를 비교합니다.

### 2. 완전성 확보

* 결측률이 80% 이상인 feature를 제거합니다.
* 야간 조건에서 `일조(hr)`, `일사량` 결측값을 0으로 대체합니다.
* 잔여 결측 row를 제거합니다.

### 3. 유효성 확보

* `발전일자`를 datetime 형식으로 변환합니다.
* 문자열 컬럼의 공백을 제거합니다.
* 주요 수치형 컬럼을 numeric 형식으로 변환합니다.
* 물리적 허용 범위에 맞지 않는 데이터를 제거합니다.

적용한 주요 기준은 다음과 같습니다.

* 설비용량(MW) > 0
* 발전량(MWh) >= 0
* 기온: -30 ~ 50
* 습도: 0 ~ 100
* 풍속 >= 0
* 적운량(10분위): 0 ~ 10
* 적운량(3분위): 0 ~ 3
* 일조(hr): 0 ~ 1
* 대기권밖일사량계산값 >= 0
* 일사량 >= 0

### 4. 일관성 확보

* 대기권밖일사량계산값이 0이고 일사량이 0인 경우를 야간 조건으로 판단합니다.
* 해당 조건에서 발전량(MWh)을 0으로 보정합니다.

### 5. 정확성 확보

* 설비용량 대비 과도한 발전량 이상치를 제거합니다.
* 기준은 `발전량(MWh) <= 설비용량(MW)`입니다.
